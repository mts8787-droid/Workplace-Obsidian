# 🚀 Antigravity 기반 Obsidian 관리 스킬 (MD변환/분류/태깅)
> **Version 1.2 업데이트 릴리즈 노트**: 
> 
> **1. 태그와 메타데이터가 훨씬 똑똑해졌습니다! (`/tag`, `/hotkey`)**
> - **태그 택소노미(Taxonomy) 전면 재정비**: 단순히 흩어져 있던 단어들을 모아, `#type`, `#market`, `#product`, `#ad` 등 15개의 명확한 카테고리(네임스페이스) 체계로 완전히 재구조화했습니다.
> - **복수 태그 허용 및 확장**: 한 문서에 관련된 시장이나 제품이 여러 개일 경우 억지로 하나만 고르지 않고, `#market/eu`, `#market/na` 처럼 필요한 만큼 여러 개(복수 태그)를 달 수 있도록 규칙을 유연하게 신설했습니다.
> - **알아서 배우고 진화하는 태그 사전**: 에이전트가 문서를 읽고 기존 태그 사전에 없는 새로운 내용이라면 알아서 새 태그를 만들고, 앞으로 계속 쓸 수 있도록 사전 파일 자체를 업데이트합니다.
> - **시스템 오류 해결**: 그동안 작동하지 않던 올인원 `/hotkey` 명령어와 폴더 정리 규칙을 완벽하게 복구했습니다.
> 
> **2. PPT 문서 변환 기능이 압도적으로 좋아졌습니다! (`/MD-PPT`)**
> - **다양한 슬라이드 레이아웃 맞춤형 파싱 룰 대거 추가**: 단순히 글자만 빼내는 게 아니라, 슬라이드의 생긴 모양을 보고 구조를 완벽히 살려냅니다.
>   - **로드맵/WBS**: 연도와 분기별 일정표 완벽 변환
>   - **공유 헤더 프로세스 표**: 위아래가 연결된 복잡한 도식 합체
>   - **보이지 않는 표(Invisible Grid)**: 테두리 없이 도형 간격만으로 만들어진 표 인식
>   - **유첨(Appendix) 예외 처리**: 가짜 거버넌스(핵심 메시지) 오류 방어
>   - **부유 주석(Floating Annotations)**: 흩어진 말풍선과 뱃지를 원래 표 칸(Cell) 안으로 알아서 귀속
> - **변환 중 튕김 현상 해결**: PPT에 복잡한 특수기호나 이모티콘이 있어도 튕기지 않도록 파이썬 내부 코드를 튼튼하게 보완했습니다.
> 
> **3. 폴더 정리를 AI가 알아서 돕습니다! (`/moc-set-user`, `/moc-set-ai`)**
> - **내 맘대로 폴더 자동 정리**: 기존에 쓰시던 폴더 구조를 옵시디언 분류 체계에 맞춰 쏙쏙 넣어주는 기능과, AI가 옵시디언 전체를 스캔해서 내 업무 스타일에 딱 맞는 새로운 폴더 트리를 추천해 주는 기능이 추가되었습니다.
> 
> **4. 이제 누구와도 시스템을 공유할 수 있습니다! (System)**
> - **PC 환경 제약 해제**: 제 로컬 PC(ts.moon) 경로에만 묶여있던 코드들을 모두 수정했습니다. 이제 이 옵시디언 폴더를 USB나 클라우드로 다른 동료에게 공유해도 아무 에러 없이 똑같이 작동합니다.

## 2. 설치순서
1. 회사 시스템을 통해 배포된 `agent-d2c.zip` 파일을 다운로드하고 압축을 풉니다.
2. 압축을 푼 폴더의 이름을 `.agent`로 변경합니다.
3. 이름을 변경한 `.agent` 폴더를 본인의 옵시디언 볼트(Vault) 최상위(Root) 경로에 복사하여 넣습니다.
4. Antigravity 에이전트에서 'Open Folder'를 통해 방금 적용한 옵시디언 볼트를 작업 공간으로 엽니다.
5. 아래 텍스트를 복사해서 Antigravity 대화창에 전송하세요.
  "내 옵시디언 최상위 폴더 하위의 `.agent` 폴더에 있는 마스터 허브 문서 `Antigravity_Agent.md`를 열고, 이 문서 안에 포함(@include)된 모든 skill, rule, hook 폴더의 하위 문서들까지 전부 스캔하고 향후 작업 시 KI에 숙지해 줘. `.agent/` 폴더 하위의 MD파일은 코어 시스템인 'Knowledge Item(KI)'에 영구적으로 등록해"


## 3. 공식 슬래시 명령어 (Slash Commands)
에이전트 연동이 완료되면, 채팅창에 아래 명령어를 입력해 빠르고 정확하게 자동화 스킬을 발동할 수 있습니다.
- `/route` : 문서들을 알맞은 폴더로 자동 라우팅
- `/tagging` : 문서의 메타데이터 빈칸 채우기 및 15개 택소노미 기반 다차원 태그 할당
- `/md` : 이메일/웹 클리핑 등 HTML 원시 파일 마크다운 정제 및 자동 태깅
- `/MD-PPT` : PPT 문서의 핵심 파싱, 보이지 않는 그리드(Invisible Grid) 표 병합, 부유 주석 귀속 및 복원
- `/title` : 본문 내용(description) 기반으로 알맞은 타이틀 및 파일명 자동 변경
- `/link` : 볼트 내 고아 문서를 찾아 기존 문서들과 문맥 연결 (위키링크)
- `/rollup` : 오늘 생성된 문서와 진행 중인 태스크를 모아 데일리 리포트 작성
- `/hotkey` : 태깅, 제목 변경, 링크 연결, MOC 갱신을 한 번에 실행하는 올인원 스킬
- `/moc-set-user` : 사용자의 기존 폴더 트리를 분석해 조니 데시멀 하위 트리로 재분배
- `/moc-set-ai` : 볼트 전체 분석 후 사용자 최적화 1-Depth 조니 데시멀 MOC 구조 재설계 및 제안
- `/harness [내용]` : 에이전트에게 새로운 룰/스킬/훅을 생성해서 영구 등록하라고 지시
