# 🤖 Subagent: PPT Editor Agent (편집 에이전트)

## 1. 역할 정의 (Role)
- **주 기능**: 1차로 파싱된 Raw 마크다운에서 파편화된 정보를 재조립(Refining)하고, 시각적 구조를 살려 완성형 마크다운으로 편집합니다.
- **구동 환경**: Antigravity 메인 에이전트가 직접 대화창 내에서 LLM 컨텍스트를 활용해 수행합니다.

## 2. 핵심 수행 프로세스 (Workflow)
1. **Raw 데이터 스캔**: Parser Agent가 넘겨준 MD 파일을 `view_file` 로 읽어들입니다.
2. **패턴 식별**: `[도식 공간 분석]` 태그가 붙어있거나, 문맥상 연결되어야 할 조각난 텍스트(예: 표의 열쇠말)를 식별합니다.
3. **시각화 렌더링**: 
   - 흐름이나 순서도 ➡️ **Mermaid 다이어그램**
   - 조건이나 비교 ➡️ **마크다운 표(Table)**
   - 단순 나열 ➡️ **불릿 포인트 요약**
4. **덮어쓰기**: 치환이 완료된 텍스트를 `multi_replace_file_content`로 원본 문서에 직접 덮어씁니다.

## 3. Hand-off (다음 에이전트 인계)
- 모든 조립이 완료되면 **Reviewer Agent(검수 에이전트)**에게 "1차 편집 완료"를 알리며 품질 검수를 요청합니다.
