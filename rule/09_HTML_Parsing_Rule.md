# 📋 Rule: HTML 정밀 파싱 및 정규화 세부 규칙

## 1. HTML 파싱 및 텍스트 추출 룰
- `<html>`, `<body>`, `<div>`, `<style>`, `<span>` 등 레이아웃용 HTML 태그를 모두 삭제합니다.
- `<p>`, `<br>` 태그는 마크다운의 줄바꿈으로 변환합니다.
- `<a href="URL">텍스트</a>` 형태의 링크는 마크다운 문법인 `[텍스트](URL)`로 안전하게 치환합니다.

## 2. 이메일 헤더 정규화 룰
- 파일 상단의 '보낸 사람', '받는 사람', '제목', '받은시간' 등의 메타 텍스트를 깔끔한 블록인용구(`>`)나 헤딩(`##`)으로 시각화하여 정보의 위계를 부여합니다.
