# 🛠️ Skill: 고아 문서 구출 및 시맨틱 링크 스킬 (/link)
**Precedence: 3**

## 1. 목적
볼트 내에 양방향 링크가 하나도 없는 고립된 문서(고아 문서)를 구출하는 것을 넘어, **TF-IDF(단어 빈도-역문서 빈도) 유사도 알고리즘**을 활용하여 문서 간의 맥락을 분석하고, 가장 연관성 높은 문서를 최대 15개까지 찾아내어 자동으로 지식 그물망을 촘촘하게 구축합니다.

## 2. 작동 트리거 (Trigger)
- **명령어**: `/link` (현재 문서 대상 연관 문서 추천)
- **일괄 적용**: `/link --folder` (대상 폴더 전체 문서를 상호 교차 분석하여 시맨틱 링크망 일괄 구축)

## 3. 고도화된 연관 문서 매칭 로직 (TF-IDF 기반)
1. **분석 풀(Pool) 확대**: 단순한 대분류(D2C, MBR)뿐만 아니라, 아래와 같은 **60여 개의 고도화된 기술/전략 키워드**를 탐지합니다.
   - `GEO Agent`, `에이전틱 커머스`, `AI Overview`, `LLM`, `Akamai`, `Semrush`, `Schema`, `AEO`, `EdgeWorker`, `VideoObject`, `RAG`, `KAG`, `Pull-based`, `CDN`, `PDP`, `ChatGPT`, `Gemini`, `Perplexity`, `서브레딧`, `시딩(Seeding)`, `Organic Traffic`, `콘센트릭스` 등.
2. **TF-IDF 유사도 스코어링**: 각 문서가 가진 고유하고 희귀한 키워드의 가중치(TF-IDF)를 계산하여, 단순 매칭이 아닌 **맥락적 연관성이 가장 높은 핵심 문서 5개**만을 엄선하여 추천합니다.
3. **자동 링크 주입**: 문서의 최하단에 `### 🔗 연관 문서 (Semantic Links)` 헤더를 생성하고, 계산된 상위 5개의 문서 링크(`[[문서명]]`)를 자동으로 주입합니다.

## 4. 결과물 출력 포맷
문서의 최하단에 다음과 같은 콜아웃 블록을 생성하여 지식을 엮어냅니다.
```markdown
> [!info] 🔗 자동 추천 연관 문서 (Semantic Linked)
> - [[관련 문서 1]] (관련 키워드: 희귀단어A, 희귀단어B...)
> - [[관련 문서 2]] (관련 키워드: 핵심단어C...)
```
