# 🛠️ Skill: 타이틀 자동 변경 스킬 (Title Change Skill)
**Precedence: 3**

## 1. 목적
문서의 `description`(요약문)이나 본문을 분석하여, 내용에 가장 알맞은 직관적이고 구조화된 타이틀(제목)을 자동으로 생성하고 변경합니다.

## 2. 작동 트리거 (Trigger)
- **사용자 슬래시 명령어**: `/title` (또는 `/title --folder`로 일괄 적용)

## 3. 워크플로우 (Workflow)
1. **문맥 분석**: 지정된 문서의 `description` 필드를 최우선으로 읽고 핵심 주제를 파악합니다.
2. **타이틀 생성**: 문서의 성격에 걸맞게 명확하고 간결한 제목을 새로 작명합니다.
3. **업데이트 적용**:
   - 프론트매터의 `title` 필드를 새로운 이름으로 갱신합니다.
   - (필요 시 파일 시스템 상의 문서 파일명(.md) 자체도 새 타이틀에 맞게 변경하도록 사용자에게 안내하거나 직접 수행합니다.)
