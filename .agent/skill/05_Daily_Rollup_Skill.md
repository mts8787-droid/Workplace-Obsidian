# 🛠️ Skill: 데일리/위클리 롤업 스킬 (Daily/Weekly Rollup Skill)
**Precedence: 3**

## 1. 목적
당일 또는 이번 주 동안 수집된 파편화된 메모와, 현재 진행 중인 태스크들을 하나의 '데일리 노트' 또는 '위클리 노트'로 끌어모아(Roll-up) 요약 리포트를 작성합니다.

## 2. 작동 트리거 (Trigger)
- **사용자 슬래시 명령어**: `/rollup` (당일/해당 주간 요약 수행)

## 3. 워크플로우 (Workflow)
1. **데이터 수집 (Fetch)**:
   - 오늘(또는 이번 주) 생성되거나 내용이 추가된 `00. Inbox` 내부의 메모들을 스캔합니다.
   - 볼트 전체에서 `status: in_progress` 상태인 파일이나 미완료된 `TODO` 항목을 추적합니다.
2. **데이터 가공 (Process)**:
   - 수집된 내용들을 '새로운 아이디어/수집품', '진행 중인 프로젝트 현황', '완료된 작업' 등의 논리적 카테고리로 분류합니다.
3. **리포트 작성 (Publish)**:
   - `01. Daily Notes` 하위에 오늘 날짜(예: `YYYY-MM-DD.md`)의 노트를 열거나 신규 생성합니다.
   - 가공된 데이터를 양방향 링크(`[[ ]]`)를 적극 활용하여, 클릭하면 즉시 원본으로 이동할 수 있는 가독성 높은 요약 리포트로 작성합니다.
