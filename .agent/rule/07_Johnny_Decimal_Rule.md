# 참조용 원본 폴더트리 (Johnny Decimal Baseline)

> **💡 시스템 노트**: 
> 이 문서는 사용자의 옵시디언 볼트를 지탱하는 **핵심 폴더 구조(Johnny Decimal) 룰베이스**입니다. 
> `/route` 등의 에이전트 스킬이 발동될 때, 이 문서를 참조하여 문서의 태그와 문맥을 스코어링하고 가장 적합한 하위 폴더(2~3 Depth)를 찾아내어 정밀 배치합니다.
> 폴더 구조가 변경되거나 확장될 경우, 반드시 이 문서를 최신화해야 에이전트가 길을 잃지 않습니다.
